# scan-book

https://scan-book.netlify.app/  
書籍のISBNバーコードをカメラでスキャンして、Googleスプレッドシートに書籍の情報を保存するアプリです

## 注意点

- iOSの場合、Safariでないとカメラが起動しません
- ログイン時にGoogleに検証されていないアプリという旨の警告が表示されます
